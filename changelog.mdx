## Upgrading OlympiTech

Follow these steps to upgrade safely. Always back up your database and `config.json` first.

<Steps>
  <Step title="Backup Data" icon="database">

    Create backups:

````bash
cp -r /path/to/olympitech/data /backup/olympitech-$(date +%Y%m%d)
cp config.json /backup/config.json
````

  </Step>
  <Step title="Update Package" icon="download">

    Use your package manager:

    <CodeGroup tabs="npm,yarn">
    ````bash
npm install olympitech@latest --save
````
    ````bash
yarn add olympitech@latest
````
    </CodeGroup>

  </Step>
  <Step title="Migrate Config" icon="settings">
    If upgrading from v0.x, convert YAML to JSON:

    <CodeGroup tabs="Before,After">
    ````json
    // config.yaml (old)
    discord:
      token: YOUR_TOKEN
      guild: 123456789
    wiki:
      db: postgres://user:pass@localhost/olympitech
    ````
    ````json
    {
      "discord": {
        "token": "YOUR_TOKEN",
        "guild": "123456789"
      },
      "wiki": {
        "db": "postgres://user:pass@localhost/olympitech"
      }
    }
    ````
    </CodeGroup>
  </Step>
  <Step title="Restart Service" icon="refresh-cw">
    Restart your OlympiTech instance:

````bash
systemctl restart olympitech
# or
pm2 restart olympitech
````

    Verify with `curl https://your-server.com/health`.

  </Step>
</Steps>

<Callout kind="alert">
  For v1.2.0+, ensure your Discord bot has `applications.commands` scope. Regenerate token if needed.
</Callout>
