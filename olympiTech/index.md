## Local Development

To run the documentation locally:

```bash
# Install dependencies
pip install mkdocs mkdocs-material

# Start the development server
mkdocs serve
```

## Building for Production

```bash
# Build the static site
mkdocs build

# Deploy to GitHub Pages
mkdocs gh-deploy
```
