In version 0.2.0+, JanusMCD has split its configuration into multiple files for better organization. All configs live in `plugins/JanusMCD/`.

## `discord.yml` (Core)
Governs the bot connection, channel IDs, and basic status features.

- **`token`** (string): Your Discord Bot Token.
- **`invite-url`**: The invite link for your Discord server.
- **`allowed-guilds`**: List of Guild IDs the bot is allowed to join (Security).
- **`channel-ids`**: List of channel IDs where **Chat Relay** happens.
- **`console-channel-ids`**: Channels where you can execute console commands.
- **`cross-discord-sync`** (bool): If true, chat from one relay channel is sent to others.

## `synchronization.yml`
Controls how data is synced between platforms.

### Ban Synchronization
- **`bans.sync-to-discord`**: Minecraft bans -> Discord bans.
- **`bans.sync-to-minecraft`**: Discord bans -> Minecraft bans.
- **`bans.sync-broadcast`**: Announce sync actions in chat.

### Other Syncs
- **`roles`**: Map Vault Groups to Discord Roles.
- **`nicknames`**: Sync Minecraft usernames to Discord nicknames.
- **`advancements`**: Post achievements to Discord.
- **`death-messages`**: Post death messages to Discord.

## `security.yml`
Protects your server from threats.

### VPN Protection
Blocks players using VPNS/Proxies to bypass bans.
- **`vpn-protection.enabled`**: Default `true`.
- **`providers`**: Supports `proxycheck` (Default), `vpnapi`, `ipqualityscore`, `iphub`.

### IP Monitoring
- **`ip-monitoring`**: Detects rapid IP switching (suspicious activity).

### Anti-Spam (Chat)
- **`anti-spam.max-messages`**: Rate limits for chat.
- **`connection-throttle`**: Prevents join-spam bots.

## `chat.yml`
Controls the Smart Filtering engine.

- **`filtering.enabled`**: global switch.
- **`banned-words`**: List of strict words to block.
- **`banned-phrases`**: List of phrases to block (smart matching).
- **`banned-patterns`**: Regex patterns.
- **`action`**: `block` (prevent sending), `replace` (stars), or `log` (silent notify).
- **`block-urls`**: Automatically remove links from chat.

## `account-linking.yml`
Manages the linking (authentication) flow.

- **`limits.max_links_per_discord`**: How many MC accounts one Discord user can have.
- **`require-verification`**: If true, users might need to verify via DM.

## `voice.yml`
Proximity Voice Chat configuration.

- **`lobby-channel-id`**: The "Waiting Room" voice channel.
- **`category-id`**: Where usage voice channels are created.
- **`proximity-radius`**: How far you can hear people (in blocks).
