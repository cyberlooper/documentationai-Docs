## Prerequisites

<Callout kind="info">

Before starting, ensure you have:

- A Discord server for your Minecraft community
- Access to a web browser
- Minecraft plugin details ready (name, version, description)

</Callout>

## Create Your OlympiTech Account

Sign up at `https://dashboard.example.com/register` to get started. OlympiTech provides a free tier for small communities.

1. Enter your email and create a strong password.
2. Verify your email via the link sent to your inbox.
3. Log in and complete the onboarding wizard by selecting "Minecraft Plugin Wiki".

<Callout kind="tip">

Save your `{API_KEY}` from the dashboard settings. You'll need it for Discord integration.

</Callout>

## Set Up Your First Plugin Wiki Page

Navigate to the dashboard wiki editor at `https://dashboard.example.com/wikis/new`.

<Steps>
  <Step title="Choose Template" icon="book-open">
    Select the "Minecraft Plugin" template. This includes sections for commands, permissions, and configuration.
  </Step>
  <Step title="Enter Plugin Details" icon="settings">
    Fill in the form:

    | Field       | Example Value              |
    |-------------|----------------------------|
    | Name        | EssentialsX                |
    | Version     | 2.20.1                     |
    | Description | Core plugin for permissions and economy. |

    Preview changes in real-time.
  </Step>
  <Step title="Publish Wiki" icon="rocket">
    Click "Publish". Your wiki is now live at `https://docs.example.com/wikis/essentialsx`.
  </Step>
</Steps>

## Connect Discord Bot for Notifications

OlympiTech's Discord bot keeps your community updated on wiki changes. Use the dashboard to generate an invite link.

<Tabs>
  <Tab title="Self-Hosted Bot" icon="server">
    Install the bot via npm:

    <CodeGroup tabs="npm,yarn">
    ````bash
    npm install olympitech-discord-bot
    ````
    ````bash
    yarn add olympitech-discord-bot
    ````
    </CodeGroup>

    Configure with your `{API_KEY}`:

    ````javascript
    const bot = new OlympiTechBot({
      apiKey: 'YOUR_API_KEY',
      guildId: 'YOUR_DISCORD_GUILD_ID'
    });
    bot.login('YOUR_BOT_TOKEN');
    ````

    Invite the bot using the generated link from dashboard.
  </Tab>
  <Tab title="Hosted Bot" icon="cloud">
    From dashboard, click "Connect Discord" and authorize OlympiTech bot directly. No code required.
  </Tab>
</Tabs>

Test the connection by editing your wiki page. Notifications should appear in your Discord `#plugin-updates` channel.
