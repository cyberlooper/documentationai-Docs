## Overview

OlympiTech empowers Minecraft plugin developers to create dynamic wikis with full Discord integration. You build comprehensive documentation for your plugins while keeping your community engaged through real-time updates in Discord channels. Say goodbye to static README filesâ€”OlympiTech syncs edits, discussions, and releases automatically, fostering collaboration between developers and players.

This platform handles everything from markdown-based wiki pages to plugin version tracking, all tied to your Discord server. Developers benefit from version control, search optimization, and community feedback loops, making plugin maintenance effortless.

## Quick Start

Get up and running in minutes. Follow these steps to launch your first OlympiTech wiki.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Create an account at `https://olympitech.com` using your Discord account for instant OAuth integration.
  </Step>
  <Step title="Create Wiki" icon="book-open">
    Select "New Wiki" and enter your plugin name, like "EssentialsX". Choose a Discord server to link.
  </Step>
  <Step title="Configure Sync" icon="settings">
    Set up webhooks in Discord. Use this example configuration:
    
````yaml
webhooks:
  updates: "https://discord.com/api/webhooks/YOUR_WEBHOOK_ID/YOUR_WEBHOOK_TOKEN"
  discussions: "https://discord.com/api/webhooks/YOUR_DISCORD_WEBHOOK"
````
    
    Replace placeholders with your values.
  </Step>
  <Step title="Publish First Page" icon="upload">
    Write your plugin overview in markdown and hit publish. Watch it appear in your Discord `#plugin-updates` channel.
  </Step>
</Steps>

## Example Integration

See how OlympiTech syncs with a Minecraft plugin config. This YAML example triggers Discord notifications on wiki edits.

<CodeGroup tabs="YAML Config,Java Listener">
````yaml
# config.yml for your Bukkit plugin
olympitech:
  enabled: true
  api-url: "https://api.olympitech.com/v1"
  wiki-id: "your-wiki-slug"
  discord-webhook: "https://discord.com/api/webhooks/YOUR_ID/YOUR_TOKEN"
````

````java
// Plugin listener example
@EventHandler
public void onWikiUpdate(WikiUpdateEvent event) {
    OlympiTechAPI.sendDiscordUpdate(event.getWikiId(), "New changes published!");
}
````
</CodeGroup>
